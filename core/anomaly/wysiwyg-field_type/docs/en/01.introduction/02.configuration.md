### Configuration

Below is the full configuration available with defaults values:

    "example" => [
        "type"   => "anomaly.field_type.wysiwyg",
        "config" => [
            "default_value" => null,
            "configuration" => "default",
            "line_breaks"   => false,
            "sync"          => true,
            "height"        => 500,
            "buttons"       => [],
            "plugins"       => [],
        ]
    ]
