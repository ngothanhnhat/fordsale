### Configuration

Below is a list of available configuration with default values:

    "example" => [
        "type"   => "anomaly.field_type.file",
        "config" => [
            "default_value" => null,
            "folders"       => [],
            "max"           => null,
            "mode"          => "default",
        ]
    ]
